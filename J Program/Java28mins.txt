Java tutorial:

Variable:
   var whose value is changing over the life time of the program.
type of var is followed by the name of var.

type name = value;
int num = 10;

types of variables:

refer: https://www.geeksforgeeks.org/variables-in-java/

local variable:
 presence inside of a block, function or method. initialization important
int a =10;

instance variable:
 presense inside a class outside a method. initialization not imp. default value 0.
we can use access modifiers.
public int a; ->0
public int a = 10;

static variable:

same as instance variable we use static keyword.
it is fixed value.
 public static int a = 10;


default value of int 0, boolean false, object null.
